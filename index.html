Java Estática Forte Orientada a objetos James Gosling, Mike Sheridan e Patrick Naughton 1991 Sun Microsystems Estados Unidos Web (backend), mobile, desktop, jogos Moderada considerando que é mais rigorosa Aumenta à medida que você explora conceitos como orientação a objetos, interfaces, coleções, exceções e tópicos avançados, como concorrência e manipulação de arquivos. Linguagem de programação de tipagem forte e estática, onde a curva de aprendizagem é maior, focada no backend
PHP Dinâmica Fraca Multiparadigma (procedural, orientada a objetos) Rasmus Lerdorf 1995 Canadá Web Relativamente fácil, especialmente para desenvolvimento web Aumenta quando você trabalha em aplicativos mais complexos Linguagem doida, onde muitos misturam o código com a estrutura do site.
Python Dinâmica Fraca Multiparadigma (orientada a objetos, funcional) Guido van Rossum 1991 Holânda Web (frontend/backend), análise de dados, intelignca artificial, automação Suave devido à sua sintaxe legível Aumenta à medida que você explora conceitos avançados, como manipulação de dados, programação funcional, criação de classes e desenvolvimento web com frameworks como Django ou Flask. Uma linguagem de programação onde a curva de aprendizagem é baixa, sua estrutura é chata no começo, mas ensia a realizar um código organizado.
C# Estática Forte Orientada a Objetos Anders Hejlsberg (liderou) 2000 Microsoft Estados Unidos Desktop, jogos, web (backend) Moderada, especialmente para iniciantes com pouca experiência em programação orientada a objetos Aumenta à medida que você explora tópicos como programação orientada a objetos avançada, manipulação de eventos, uso de bibliotecas de terceiros e desenvolvimento de aplicativos desktop ou jogos Uma linguagem derivada do C muito ultilisada no desenvolvimento de jogos.
Lua Dinâmica Fraca Multiparadigma (imperativa, procedual, orientada a objetos, funcional) Roberto Ierusalimschy, Luiz Henrique de Figueiredo e Waldemar Celes 1993 Brasil Script para jogos, sistemas embarcados Fácil de aprender, especialmente para scripting Cresce quando você explora recursos avançados, como manipulação de tabelas, programação orientada a objetos em Lua e incorporação de Lua em aplicativos maiores Linguagem de programação muito parecida coom python, que é brasileira.
